Chord Debug Properties

This section describes properties that specify the amount of debug information that Chord must produce during execution.

  • chord.verbose
    Type: bool
    Description: Produce more verbose output during Chord's execution.
    Default value: false

  • chord.bddbddb.noisy
    Type: bool
    Description: Produce more verbose output during the Datalog solver bddbddb's execution.
    Default value: false

  • chord.save.maps
    Type: bool
    Description: Write to file [chord.bddbddb.work.dir]/<...>.map when saving program domain named <...>.
    Default value: true
    Note: This functionality is useful for debugging Datalog programs using the debug target provided in file build.xml in Chord's main/ directory (see Section ).
